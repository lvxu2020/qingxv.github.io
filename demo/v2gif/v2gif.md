# 产品概述 <a id="产品概述-1"></a>
win64上的一款qt写的软件，利用ffmpeg将视频文件转化成gif动图。
## 功能简介 <a id="功能简介-1"></a>
1.选择取视频数据流信息预览（视频的宽、高、帧率等）  
2.自定义视频截取时间  
3.帧率可调节  
4.比例缩放（根据宽度设定的值按比例缩放）
